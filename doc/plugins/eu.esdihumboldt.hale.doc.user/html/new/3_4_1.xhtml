What's new in hale studio 3.4.1?

This is a bugfix-only release that does not contain new features. You can find a list of the most important fixes below.For a full list, please check out the changelog on GitHub

  • Fixed problem with AppSchema export configuration dialog on Windows
  • Updated cached GML XSD to version 3.2.2
  • Loading an Excel lookup table multiple times no longer causes an error
  • Fixed file name problem when partitioning by feature type in a GML export
  • Fixed problem that prevented partitioning by feature type when transforming external data or with CLI

The majority of the new features and improvements were made possible through our contracts with more than 30 customers. We'd like to thank you all for your support!
